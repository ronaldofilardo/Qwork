# Documentação Arquivada

## Estrutura

| Subpasta                | Conteúdo                                                 |
| ----------------------- | -------------------------------------------------------- |
| `build-approvals/`      | Aprovações de build históricas (one-off)                 |
| `deployments/`          | Instruções e confirmações de deploys passados            |
| `diagnosticos/`         | Diagnósticos pontuais de bugs já resolvidos              |
| `reports/`              | Relatórios de auditorias e remoções concluídas           |
| `analises/`             | Análises técnicas de problemas já corrigidos             |
| `security/`             | Documentos de segurança obsoletos (ex: padrões Supabase) |
| `correcoes-historicas/` | Correções pontuais já aplicadas                          |
| `corrections-2025/`     | Correções de dezembro 2025                               |
| `implementacoes-2024/`  | Implementações de 2024/2025                              |
| `prs/`                  | Descrições de PRs históricas                             |

- Documentos aqui **NÃO devem ser referenciados** por código ativo
- Para encontrar a versão atual de qualquer doc, consulte [docs/README.md](../README.md)
